# GPU 噪声生成器

在编辑器内使用 GPU Compute Shader 实时生成程序化噪声贴图，支持 8 种噪声类型、法线贴图导出、无缝平铺及多种高级效果。

---

## 快速开始

1. 打开 **XY 编辑器工具集**，在左侧选择 **GPU 噪声生成器**
2. 在 **预设** 区域一键应用内置效果（云层、大理石、地形等），或手动调参
3. 右侧实时预览满意后，点击 **保存 PNG** 导出灰度贴图，或 **保存法线** 导出法线贴图
4. 贴图自动导入到 Project 窗口的指定保存路径

---

## 噪声类型 & 适用场景

| 类型 | 效果 | 推荐用途 |
|------|------|----------|
| **Perlin** | 平滑连续 | 地形高度图、云层、烟雾 |
| **Simplex** | 与 Perlin 相似但更快 | 大面积实时生成时优先选此 |
| **Worley** | 蜂窝/细胞形态 | 石材纹理、鳞片、细胞 |
| **Value** | 偏方块状 | 像素风、低频遮罩 |
| **FBM** | 多层叠加分形 | 复杂地形、逼真云彩 |
| **Ridged** | 尖锐山脊 | 山脉、岩石裂缝、闪电 |
| **Turbulence** | 扭曲流动 | 火焰、烟雾、水流 |
| **VoronoiEdge** | 细胞边界线 | 龟裂纹、马赛克边缘 |

## 参数说明

### 基础参数

| 参数 | 说明 |
| --- | --- |
| **Resolution** | 输出贴图分辨率（建议 2 的幂次，64 ~ 2048） |
| **Scale** | 噪声频率，值越大纹理越细碎 |
| **Offset X/Y** | 平移采样坐标，用于生成不同区域或制作滚动动画 |
| **Seed** | 随机种子，相同种子输出完全相同 |
| **Seamless** | 勾选后生成可无缝平铺的纹理 |
| **Invert** | 黑白反转 |
| **Contrast / Brightness** | 对比度与亮度微调 |

### FBM / Ridged / Turbulence 额外参数

| 参数 | 说明 |
| --- | --- |
| **Octaves** | 叠加层数（1~8），越多细节越丰富 |
| **Persistence** | 每层振幅衰减系数（通常 0.5） |
| **Lacunarity** | 每层频率增长系数（通常 2.0） |

## 高级功能

### 域变形（Domain Warping）

用噪声扭曲采样坐标，生成有机流动效果（大理石、木纹、熔岩）。

- **变形强度** — 扭曲幅度（0~2）
- **变形缩放** — 扭曲噪声的频率
- **迭代次数** — 嵌套层数（1~4，越多越复杂）

### 渐变映射（Gradient Mapping）

将灰度噪声重新着色：指定一张 1×N 渐变纹理，黑色噪声区域映射到渐变左侧颜色，白色映射到右侧。适合彩色地形、热力图。

### 噪声混合（Noise Blending）

将当前噪声与另一张贴图叠加，支持 6 种混合模式：Mix / Add / Multiply / Screen / Overlay / Difference。

### 预设系统

内置 6 种预设（Clouds / Marble / Terrain / Cells / Wood / Fire），点击即可一键载入参数。也可将当前配置 **保存为自定义预设**，下次直接加载。

### 法线贴图输出

点击 **保存法线** 按钮，根据当前噪声高度自动生成法线贴图，`Normal Strength` 控制凹凸强度。

## 使用技巧

- **调参效率** — 预览时用小分辨率（256），确认效果后再切换到目标分辨率导出
- **无缝平铺** — Scale 使用整数值时更容易获得无缝结果
- **山脉地形** — 选 Ridged，Octaves 设 6~8
- **火焰 / 烟雾** — 选 Turbulence，逐帧递增 Offset Y 可制作滚动动画序列帧
- **裂纹龟裂** — 选 VoronoiEdge，勾选 Invert，拉高 Contrast
- **大理石纹** — 选 FBM + 开启域变形，Iterations 设 2~3

---

## 注意事项

- 需要设备支持 **Compute Shader**（DirectX 11 / Metal / Vulkan）
- 分辨率超过 2048 时生成耗时明显增加，建议按需使用
- EXR 格式保留完整精度，适合用作高度图或流体模拟输入
