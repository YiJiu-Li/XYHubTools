# 资源复制工具

将 Unity 项目中的资源（目录或单文件）快速复制到另一个位置，同时自动处理 `.meta` 文件，避免 GUID 冲突。

---

## 快速开始

1. 打开 **XY 编辑器工具集**，在左侧选择 **资源复制工具**
2. 选择复制模式：**目录模式** 或 **单文件模式**
3. 填写源路径和目标路径
4. 按需勾选过滤选项
5. 点击 **开始复制**

---

## 复制模式

### 目录模式

递归复制整个文件夹及其所有子资源。

- **源路径** — 要复制的文件夹，填写相对于项目根目录的路径（如 `Assets/Art/Characters`）
- **目标路径** — 复制到的目标文件夹（不存在时会自动创建）
- 复制完成后 Project 窗口自动刷新

### 单文件模式

只复制指定的单个资源文件及其 `.meta`。

- **源路径** — 资源文件路径（如 `Assets/Art/hero.fbx`）
- **目标路径** — 目标文件夹路径

---

## 过滤选项

| 选项 | 说明 |
| --- | --- |
| **包含代码文件** | 勾选后复制 `.cs`、`.shader` 等脚本文件，不勾选则跳过 |
| **包含 Packages 依赖** | 勾选后将 Packages 目录下被引用的资源一并复制 |
| **覆盖已存在文件** | 目标存在同名文件时是否直接覆盖，不勾选则跳过 |

---

## 进度与日志

- 复制过程中底部显示**进度条**和当前处理的文件名
- 右侧日志面板实时输出每条操作记录
- 点击 **取消** 可中断正在进行的复制
- 上次使用的路径会自动记忆，下次打开无需重新填写

---

## 注意事项

- 源路径和目标路径不能相同或存在包含关系
- 复制完成后建议检查 Console，确认无引用丢失警告
- 如需副本与原资源完全独立，复制后在 Project 窗口对目标文件夹执行 **Reimport**
